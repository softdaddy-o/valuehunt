================================================================================
  QUICK START OPTIONS
================================================================================

OPTION 1: Scripts (Recommended for local development)
---------
Windows:
  1. Configure .env file
  2. Double-click: scripts\start_services.bat
  3. Services will start in separate windows
  4. To stop: scripts\stop_services.bat

Unix/Linux/macOS:
  1. Configure .env file
  2. chmod +x scripts/*.sh
  3. ./scripts/start_services.sh
  4. To stop: ./scripts/stop_services.sh

OPTION 2: Docker Compose (Recommended for production)
---------
Any OS:
  1. Install Docker and Docker Compose
  2. Configure .env file
  3. docker-compose up -d
  4. Access: http://localhost:8000
  5. Stop: docker-compose down

OPTION 3: Manual (For development/debugging)
---------
  1. Create virtual environment: python -m venv venv
  2. Activate: source venv/bin/activate (or venv\Scripts\activate on Windows)
  3. Install: pip install -r requirements.txt
  4. Start FastAPI: uvicorn app.main:app --reload
  5. Start Celery: celery -A app.celery_app worker --loglevel=info
  6. Start Beat: celery -A app.celery_app beat --loglevel=info

================================================================================
  IMPORTANT SETUP STEPS
================================================================================

1. Configure Environment (.env)
   - Set DATABASE_URL
   - Set REDIS_URL
   - Add DART_API_KEY (get from https://opendart.fss.or.kr/)
   - Add other optional API keys

2. Get DART API Key
   - Visit: https://opendart.fss.or.kr/
   - Register for free account
   - Generate API key
   - Add to .env: DART_API_KEY=your-key

3. Choose Deployment Option
   - Scripts: For local development
   - Docker: For production/distribution
   - Manual: For debugging
